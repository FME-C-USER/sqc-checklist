## 2. `點檢人員`（常駐主檔）

| 欄位 | 說明 | 範例 |
|------|------|------|
| 員編 | 唯一鍵 | `A001` |
| 姓名 | | 王小明 |
| 部別 | | 營運一部 |
| 課別 | 點檢名單依此比對帶出 | 北一課 |
| AD帳號 | **僅管理者需填**；一般點檢員可留空 | wang.ming |
| 角色 | `管理者` / `點檢員` | 點檢員 |

### 權限模型（混合）
- 登入：FME `CheckUserId` 驗證帳密（**任何有效公司 AD 皆可登入**）
- 角色判斷：以 **AD帳號** 比對 `點檢人員`
  - 比對到（通常是管理者，名冊有填 AD帳號 + 角色=管理者）→ 取其角色
  - 比對不到 → 一般 **點檢員**，登入後於基本資料**自行選取姓名/部別/課別**
- 名冊用途：提供「姓名下拉選單」與各人的部別/課別；**只有管理者需要填 AD帳號**
- **點檢員**：基本資料、點檢作業、查詢紀錄（改/刪自己上傳的）、彙總（自己課別）
- **管理者**：全部功能 + **維護專區**（上傳店鋪主檔/人員/店鋪名單/題庫）+ 全課別查詢與彙總

## 4. `題庫_{月}`（每月匯入；同檔名覆蓋、以最新為主）

| 欄位 | 說明 | 範例 |
|------|------|------|
| 排序 | 顯示順序 | 7 |
| 編號 | 題目唯一鍵 | `A7` |
| 大分類 | | 活動告示 & 清潔維護 |
| 題號名稱 | | 7.貨架清潔 |
| 配分 | 該題滿分 | 8 |
| 計分方式 | `合格0分`（整題合格或 0）／`分區扣分`（點缺失由滿分往下扣） | 分區扣分 |
| 每項扣分 | 分區扣分用，每個缺失扣幾分 | 2 |
| 子項清單 | 分區扣分用，見下方編碼規則 | `OC\|WI\|…\|其他貨架:填寫` |
| 規範說明 | 評核內容 | 貨架層板乾淨無積塵髒污… |

### 子項清單編碼規則（`分區扣分` 用，以 `|` 分隔）
- 一般子項：`OC`、`WI`、`冷凍櫃`…（每個扣「每項扣分」）
- 需扣多單位：`標籤:2`（扣 2×每項扣分）例：`馬鈴薯機+夯番麥:2`
- 可填寫子項：`標籤:填寫`（點檢人員填入名稱，填幾項扣幾個缺失）例：`其他貨架:填寫`、`其他:填寫`
- 附說明：`標籤（說明文字）` 例：`蛋鍋（不足1/2；麻辣蛋不觀察）`
